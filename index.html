Smart start

Wat houdt Smart start in?

Het doel van Smart start is om meer kennis te ontwikkelen over Smart Industry. U kunt hierbij denken aan het Smart Industry model en het bepalen van de Smart research. Tijdens dit vak heb ik ook meer geleerd over 3d-printing en het maken van een student journey

1. Beschrijft de mate van volwassenheid van een organisatie m.b.t. Smart Industry in een business case
2. Communiceert op effectieve wijze zijn bevindingen naar verschillende doelgroepen
3. Werkt actief aan zijn professionele ontwikkeling

Computervaardigheid

Tijdens deze minor heb ik veel geleerd over computers. Voor deze minor wist ik niet eens wat Visual studio code en Github was, maar ik kan nu met beide overweg. Ik heb hiermee dus een flinke vooruitgang geboekt, maar ook over de dagelijkse programma's, zoals Powerpoint en Excel ben ik meer te weten gekomen.

Doelen stellen

Omgaan met details

Aanpassingsvermogen

Tijdens het uitvoeren van het project kregen we steeds nieuwe ideeën. Hierdoor heb ik een beter aanpassingsvermogen gekregen en accepteer ik het sneller als iets moet worden aangepast.

Smart connection

Wat houdt Smart connection in?

Tijdens Smart connection heb ik heel erg veel geleerd over programmeren. In het begin wist ik niet eens was CSS en HTML waren en nu heb ik er zelf een portfolio meegecreërd. Daarnaast heb ik ook veel geleerd over big data en machine learning.

1. Past nieuwe technologieën op het gebied van web, data, cloud en AI toe op een praktijkcasus
2. Plaatst de casus in een business context

HTML

HyperText Markup Language (afgekort HTML) is een op SGML gebaseerde opmaaktaal voor de specificatie van documenten, voornamelijk bedoeld voor het wereldwijde web.

CSS

Cascading Style Sheets (afgekort tot CSS), stijlbladen, zijn een mogelijkheid om de vormgeving van webpagina's los te koppelen van hun feitelijke inhoud en centraal vast te leggen.

Javascript

Java is een objectgeoriënteerde programmeertaal. Java is een platformonafhankelijke taal die qua syntaxis grotendeels gebaseerd is op de (eveneens objectgeoriënteerde) programmeertaal C++. Java beschikt echter over een uitgebreidere klassenbibliotheek dan C++.

Smart project

Wat houdt Smart project in?

Nadat we zes weken les hadden gehad, kregen we een opdracht uit de praktijk. Bij ons kon deze helaas niet doorgaan, vanwege de coronacrisis en daarom hebben we zelf een opdracht bedacht.

1. Ontwikkelt een prototype van de oplossings(richting)
2. Draagt vanuit verschillende rollen bij aan het succes van een projectteam
3. Vertaalt een praktijkcasus naar een onderzoeksplan, voert deze uit en evalueert de kwaliteit van uitvoering
4. Draagt actief bij aan een optimale werksfeer sfeer binnen en buiten zijn team
5. ontwikkelt een professioneel netwerk
